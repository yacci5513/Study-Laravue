## 학습 구조

### 1. 프론트엔드 기초
| 폴더 | 내용 |
|------|------|
| `html/` | HTML5 기본 문법, 시맨틱 태그, 폼 |
| `css/` | CSS3 레이아웃, Flexbox, 반응형 |
| `javascript/` | ES6+, DOM 조작, 이벤트 처리 |
| `bootstrap/` | Bootstrap 컴포넌트, 그리드 시스템 |

### 2. PHP & OOP
| 폴더 | 내용 |
|------|------|
| `php/` | PHP 기본 문법, 배열, 함수 |
| `OOP/` | 클래스, 상속, 인터페이스, 다형성 |

### 3. Vue.js
| 폴더 | 내용 |
|------|------|
| `vue/vueroute/` | Vue Router를 활용한 SPA 라우팅 |
| `vue/vueshop/` | 쇼핑몰 프로젝트 (상태관리, 컴포넌트) |
| `vue/vuestagram/` | Instagram 클론 (Vue 단독 버전) |

### 4. Laravel 프레임워크
| 폴더 | 내용 |
|------|------|
| `laravel_edu/` | MVC 패턴, 라우팅, Blade 템플릿 |
| `laravel_api/` | RESTful API 설계 및 구현 |
| `jwt/` | JWT 토큰 기반 인증 시스템 |

### 5. 풀스택 (Laravel + Vue)
| 폴더 | 내용 |
|------|------|
| `laravel_vue/` | Laravel 백엔드 + Vue 프론트엔드 통합 |
| `laravel_vuestagram/` | Instagram 클론 풀스택 프로젝트 |

### 6. 실습 프로젝트
| 폴더 | 내용 |
|------|------|
| `side_project/calculator/` | 계산기 |
| `side_project/mini_board/` | 게시판 (기본 CRUD) |
| `side_project/mini_board2/` | 게시판 v2 (기능 개선) |
| `side_project/mini_multi_board/` | 다중 게시판 |
| `side_project/laravel_board/` | Laravel 게시판 |
| `side_project/laravel_music/` | 음악 스트리밍 서비스 |

### 7. 데이터 분석 (Python)
| 폴더 | 내용 |
|------|------|
| `python/01.python.ipynb` | Python 기초 문법 |
| `python/02. 데이터시각화.ipynb` | Matplotlib, Seaborn 시각화 |
| `python/03. 판다스.ipynb` | Pandas 데이터 처리 및 분석 |

## 학습 순서 (권장)

```
HTML/CSS/JS 기초 → PHP & OOP → Laravel 기본 → Vue.js
→ Laravel + Vue 통합 → 실전 프로젝트 (Vuestagram) → API & JWT 인증
```
